					!!HOW TO EXECUTE!!


1. Extract the zip file "online_education_portal".

2. Open "eclipse Java EE" and import the "online_education_portal" folder.

3. Run the "index.jsp" file under: src -> main -> webapp.

4. The admin login page will be displayed. To register new admin click on "click here to register". 
   
5. The static admin Associate ID is "12345" and password is "admin".

6. After successful login, home page will be displayed.
